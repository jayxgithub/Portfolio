🎮 CNC Controller Using MPG

📌 Overview

Microcontroller-based CNC jogging controller for precise manual axis control using an MPG encoder.

🎯 Features

  • Quadrature encoder interfacing
  • Real-time axis control
  • Debouncing & signal processing

⚙️ How It Works

  1. MPG generates pulses
  2. Microcontroller reads encoder signals
  3. Firmware processes movement
  4. Axis moves precisely
