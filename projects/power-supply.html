⚡ Dual Output Power Supply PCB

📌 Overview

This project is a custom-designed PCB using KiCad that converts 12V DC into regulated 5V and 3.3V outputs for embedded systems.

🎯 Features

  • 12V → 5V & 3.3V conversion
  • LD1117 linear regulators
  • Fuse-based input protection
  • LED indicators

⚙️ How It Works

  1. 12V input is applied through terminals
  2. Fuses protect against overcurrent
  3. LD1117 regulators convert voltage
  4. Capacitors stabilize output
